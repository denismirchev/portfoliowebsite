# Project Images

Upload your project images here!

## How to use:

1. Add your project images to this folder (`public/images/projects/`)
2. Name your images descriptively (e.g., `pigeon-project.jpg`, `saas-mvp.png`)
3. Update the `image` field in `components/sections/projects.tsx` to reference your local images

## Example:

Instead of:
```javascript
image: 'https://images.pexels.com/photos/...'
```

Use:
```javascript
image: '/images/projects/your-image-name.jpg'
```

## Supported formats:
- JPG/JPEG
- PNG
- WebP
- GIF

## Recommended image size:
- Width: 800-1200px
- Aspect ratio: 16:9 or 4:3
- File size: Keep under 500KB for best performance
